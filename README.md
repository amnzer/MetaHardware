# MetaHardware
Hardware for MetaSense's KiCad design. 

hardware_v1 is the PCB version used for the prototype mentioned in the final report. 

hardware_v2 aims to fix a few of the major problems that hardware_v1 has in terms of reliability and simplicity, although we have not yet tested hardware_v2. hardware_v2 is the more natural starting point for the next team to pick up the MetaSense project
